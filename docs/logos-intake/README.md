# LOGOS Intake Notes

This folder holds durable seed notes for LOGOS intake items that are not yet flowing through a full ingestion pipeline.

Notes are now organized by pool:

- `raw/`
- `private/`
- `public-safe/`

Use these notes when you have source metadata now, but the richer canonical/import path is still future work.

Examples:

- forum support threads
- email threads
- bug reports
- deployed-app feedback items

These notes are not canonical event history.

They are a durable bridge into project memory so humans and AI can keep working with the intake before the full source adapter exists.
