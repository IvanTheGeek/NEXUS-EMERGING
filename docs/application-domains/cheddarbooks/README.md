# CheddarBooks

CheddarBooks is a division under Cheddar focused on practical finance, recordkeeping, bookkeeping, and support-oriented tools built on top of the NEXUS and FnUI foundations.

Current active tool apps and flows include:

- [`LaundryLog`](laundrylog/README.md)

Start here:

- [`../cheddar/README.md`](../cheddar/README.md)
- [`../cheddarbooks-foundation.md`](../cheddarbooks-foundation.md)
- [`laundrylog/README.md`](laundrylog/README.md)

Current direction:

- CheddarBooks is the business-focused division inside the broader Cheddar line
- the long-term flagship goal is a free, open, privacy-conscious alternative to QuickBooks and Xero
- LaundryLog is the first concrete tool app under that area
- PerDiemLog has already surfaced as the next likely complementary tool app
- later CheddarBooks support/debugging and business flows can sit beside those tools without forcing everything into one app
