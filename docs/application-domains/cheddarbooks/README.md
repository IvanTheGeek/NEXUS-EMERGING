# CheddarBooks

CheddarBooks is an application-domain and branding area for practical finance, recordkeeping, and support-oriented tools built on top of the NEXUS and FnUI foundations.

Current active tool apps and flows include:

- [`LaundryLog`](laundrylog/README.md)

- later CheddarBooks support/debugging and business flows can sit beside it without forcing everything into one app
