# CheddarBooks LaundryLog Privacy And Ownership

This note captures the current privacy and ownership requirements for LaundryLog.

## Privacy And Ownership Requirements

LaundryLog should align with the wider app principle that users retain control of their data.

The user's personal expense data should remain theirs. LaundryLog should not assume routine sharing of that data with large companies, advertising systems, or government systems just to make the app usable.

That means the app should remain compatible with:

- local-first use
- offline entry
- optional hosted services rather than mandatory cloud dependence

## Design Consequences

This currently implies:

- the first useful product should not require constant server contact
- user data should remain understandable and portable
- later sync or convergence should not erase local ownership
- outbound sharing should be explicit and user-directed rather than automatic
- hosted features should enrich the app rather than define whether it can be used
