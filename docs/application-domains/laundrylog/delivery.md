# LaundryLog Delivery

This note captures the current delivery and runtime direction for LaundryLog.

## Delivery Requirements

Current direction:

- browser-first
- mobile-friendly
- likely PWA packaging

Optional native-host exploration may come later, but it should not be required for the first useful product.

## Current Delivery Pressure

LaundryLog is currently pushing the platform toward:

- fast mobile interaction
- installable web delivery
- offline-tolerant behavior
- small screen composition

## Deferred Or Open Delivery Decisions

Not yet settled:

- exact native-host path, if any
- exact offline storage/runtime shape
- exact hosted-service boundary for shared services
