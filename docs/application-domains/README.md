# Application Domains

This area holds domain-specific foundations for concrete systems built on top of NEXUS.

These docs are distinct from:

- NEXUS core meaning
- engineering conventions
- ingestion and LOGOS architecture
- FnHCI/FnUI platform foundations

Application-domain docs should describe the concrete system itself while still linking back to the platform lines it depends on.
