# NEXUS-Objects

This folder represents the object layer for NEXUS.

It is intended to hold:

- original provider export zips
- extracted working raw payloads
- attachments and audio
- manually added recovered artifacts
- archived local Codex session transcripts

This layer is source-preserving and supports later reparsing and reconciliation.

## Current Raw Sources

One of the first local raw-source paths in this workspace is:

- `providers/codex/`
  mirrored Codex session transcripts and session index snapshots exported from `~/.codex`
